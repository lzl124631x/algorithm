# Catalan Number

The `n`th Catalan number is given directly in terms of binomial coefficients by

$$
C_n = \dfrac{1}{n+1}{2n \choose n} = \dfrac{(2n)!}{(n+1)!n!}=\prod_{k=2}^{n}\dfrac{n+k}{k}\quad \text{for } n \ge 0
$$
